## 传统条件概率的多层概率问题

![23新1 T21](/notes/study/20250323-1-1.png)

甲乙两人偷懒, 每次由其中一人投篮, 规则如下: 若命中则此人继续投篮,  若未命中则换为对方投篮. 无论之前投篮情况如何, 甲每次投篮命中率均为 $0.6$ , 乙每次投篮命中率均为 $0.8$ . 由抽签确定第 $1$ 此投篮的人选, 第 $1$ 次投篮是甲乙概率各位 $0.5$ .

- 求第 $2$ 次投篮的人是乙的概率;
- 求第 $i$ 次投篮的人是甲的概率;
- 已知: 若随机变量 $X_i$ 服从两点分布, 且 $P(X_i = 1) = 1 - P(X_i = 0) = q_1, \ i = 1, 2, \cdots, n$ , 则 $E(\sum_{i = 1}^n X_i) = \sum_{i = 1}^n q_i$ . 记前 $n$ 次中甲投篮次数为 $Y$ , 求 $E(Y)$ .

该问题满足 `前者影响后者` 的特征, 满足马尔可夫链形式, 我们可以绘制树状图来分析:

```
开始
 ├─ 甲
 |  ├─ 甲
 |  |  ├─ 甲
 |  |  └─ 乙
 |  └─ 乙
 |  |  ├─ 甲
 |  |  └─ 乙
 └─ 乙
    ├─ 甲
    |  ├─ 甲
    |  └─ 乙
    └─ 乙
       ├─ 甲
       └─ 乙
```

第二次是乙的概率很简单, 直接使用条件概率计算即可.

记 $X_n$ 为第 $n$ 次投篮为`甲`, $Y_n$ 为第 $n$ 次投篮为`乙` , 则:

$$
P(Y_2) = P(X_1) \times \underset{甲未投中}{0.4} + P(Y_1) \times \underset{乙投中了}{0.8} \\
 \ \ = 0.5 \times 0.4 + 0.5 \times 0.8 = 0.6
$$
